🌌 Cloud Notes – Galaxy Sticky Wall

A minimalist, space-themed sticky notes web app that lets you float your thoughts into a digital galaxy ✨
Built with a focus on aesthetics and simplicity, this project transforms ordinary note-taking into a visually immersive experience.

*✨ Features*
🌠 Galaxy-Themed UI
A beautiful starfield background with glowing, floating sticky notes.
🧠 Persistent Storage
Notes are saved using localStorage, so they stay even after refresh or closing the browser.
🎨 Dynamic Sticky Notes
Each note:
Has a random tilt
Uses vibrant colors
Mimics real-life sticky notes
🧩 Responsive Grid Layout
Notes automatically adjust to screen size using CSS Grid.
⏱️ Auto Timestamping
Every note records the exact time it was created.
🧼 Clean UI/UX
Glassmorphism input box
Smooth, minimal interactions
No clutter
⚠️ Note Limit System
Maximum of 20 notes to keep your galaxy organized.
🧪 Secret additional Feature
Type "antigravity" in the input box to watch a little magic👀

*Tech Stack*
HTML5 – Structure
CSS3 – Styling (Grid, animations, starfield, glassmorphism)
JavaScript (Vanilla) – Functionality & logic
localStorage – Data persistence
